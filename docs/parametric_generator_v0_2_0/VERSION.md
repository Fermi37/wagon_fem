# Parametric Generator v0.2.0

Version identifier: `0.2.0-passenger-body-frame`

Version `0.2.0` extends the beam-only wagon generator toward passenger rolling stock. The main deliverable is a parameterized engineering description of the passenger-car body as a spatial system of connected beams suitable for subsequent finite-element calculations of member forces and bending moments.

The version keeps compatibility with `wagon_fem` combined `CSV` export. `CSV` means `Comma-Separated Values`, a tabular text format with delimiter-separated columns.
